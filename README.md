# VibraChat

### I'm using this project to learn about application security by ensuring that the data exchanged between the frontend and backend is secure from attacks.

### Frontend tools:

- Vite.js
- TypeScript
- Tailwind
- Material UI
- Redux

### Backend tools:

- Express.js
- SQlite for development
- PostgreSQL for Production
- TypeScript
- Prisma

### Data sharing tool:

- Socket.io

### Deployment tool:

- Vercel (Frontend)
- Render (Backend)
